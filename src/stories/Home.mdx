# Tanoshi design system

## Introduction

Tanoshi is a design system for [sveltekit](https://kit.svelte.dev/) web applications. It is a collection of reusable and customisable components that can be used together to build any number of applications.

## What this website is

This website was made using [storybook](https://storybook.js.org/). It will be used for documentation and as an interactive showcase for the components.

## Getting started

### Installation

```bash
npm install deozza/tanoshi
```

### Configuration

The components have no style by default to let you apply your own graphic chart.

In order to style the components, you need to add specific css variables in your main css file.

```css
:root{
    --primary: rgb(59 130 246);
    --primary-dark: rgb(37 99 235);
    --secondary: rgb(107 114 128);
    --secondary-dark: rgb(75 85 99);
    --success: rgb(34 197 94);
    --success-dark: rgb(22 163 74);
    --warning: rgb(234 179 8);
    --warning-dark: rgb(202 138 4);
    --danger: rgb(239 68 68);
    --danger-dark: rgb(220 38 38);
    --info: rgb(103 232 249);
    --info-dark: rgb(34 211 238);
    --white: rgb(255 255 255);
    --white-dark: rgb(100 116 139);
    --black: rgb(0 0 0);
    --black-dark: rgb(100 116 139);
}
```

These are the default values, but you can change them to fit your needs.

### Usage

Each component has a `*model.ts` file, to customize the component style and comportment, and a `.svelte` file containing the actual component. 

Let say you want to use the `Button` component :

```html

<script lang="ts">
    //import the component and its model :
    import { TanoshiButton, TanoshiButtonModel } from "tanoshi/atoms";

    //create the component model :
    const button: TanoshiButtonModel = new TanoshiButtonModel('Click me');

</script>

<!-- use the component : -->
<TanoshiButton tanoshiButtonModel={button} />

```

If you need to add style to your button, you could do it through the available methods of the model : 

```html

<script lang="ts">
    //import the component and its model :
    import { TanoshiButton, TanoshiButtonModel } from "tanoshi/atoms";

    //create the component model :
    const redOutlinedButton: TanoshiButtonModel = new TanoshiButtonModel('Click me')
        .setTheme('danger') //changes the color
        .setIsOutlined(true) //makes the background transparent and keeps only the border

</script>

<!-- use the component : -->
<TanoshiButton tanoshiButtonModel={button} />

```

If you need to know more about a component and how to use it, you can check each of them in the sidebar on the left.
